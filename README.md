# Stop Bored Play Board

konsent2@gmail.com이 운영하는 보드게임 일기장. Jekyll로 빌드되어 GitHub Pages(konsent.github.io)에 배포된다.

## 구조

- `_posts/short/` — 단편 후기
- `_posts/long/` — 장편 후기
- `_posts/rules/` — 규칙서 번역
- `_posts/notitles/` — 잡담
- `_posts/aar/` — 플레이 후기(After Action Report)
- `wepapps/` — 솔로 헬퍼, 워게이밍 오라클 등 독립 웹앱
- `photo/` — 게시물 이미지

## 로컬에서 실행

```bash
bundle install
bundle exec jekyll serve
```

기반 테마는 [urban-theme](http://erosie.net/)의 Jekyll 포트를 수정해 사용한다.
