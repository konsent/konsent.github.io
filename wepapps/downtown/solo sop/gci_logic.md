# GMT Downtown — VPAF 지상관제(GCI) 행동 로직 설계안 v3

GCI의 유일한 목표: **폭격기가 목표물에 폭탄을 투하하지 못하게 방해하는 것.**

미군 전력을 물량이나 화력으로 압도하는 것은 불가능하다. MiG는 치고 빠지는 방식으로 폭격 임무를 교란하는 데만 집중한다.

---

# 설계 원칙

## 원칙 1: 정보 비대칭

GCI는 **레이더 정보와 파일럿 육안 보고**만으로 판단한다.

| 알 수 있는 것 | 알 수 없는 것 (접촉 전) |
|---|---|
| 적 편대 위치(헥스) | 임무 종류 (타격/SEAD/CAP 등) |
| 고도 대역 (저/중/고) | 무장 상태 (폭장 여부) |
| 편대 규모 (기체 수) | 정확한 기종 |
| 이동 방향 및 속도 | 연료 상태 |
| 편대 배치 구조 (선두/후속) | |

**접촉 후 파일럿 보고로 추가 확인 가능:**
- 기종 (F-105인지 F-4인지 등)
- 폭장 여부 (육안으로 확인)

## 원칙 2: MiG 기종 비공개

MiG는 미군 탐지 전까지 기종이 공개되지 않는다.

- GCI는 자신이 어떤 MiG를 운용하는지 알지만, 미군 플레이어는 접촉 전까지 기종 불명
- 기종 노출을 최대한 늦추는 것이 전술적 이점
- MiG-21: 고속 접근-발사-이탈 → 노출 시간 최소화
- MiG-17: 근접 기동전 유도 → 접근 시 기종 드러나지만 이후 기동전 우위

## 원칙 3: GCI는 전체를 조율한다

GCI는 전체 레이더 화면을 보고 MiG 전력을 배분한다:
- **공격 편대**: 폭격기 방해 전담
- **견제 편대**: escort를 묶어두는 역할 (실제 교전 최소화)
- **예비 편대**: 상황에 따라 투입

# 1단계: 레이더 판독 — 폭격기 식별

이동 페이즈 시작 시, GCI는 폭격기를 찾는다.

## 위협 편대 탐지 기준

### SEAD 탐지 (최우선)

아래 패턴이면 **SEAD 가능성 있음**으로 처리:

```
소규모 편대(2~4기)가 주 공격대와 분리되어 있는가?
 └─ YES → 아래 확인

SAM 사이트 방향으로 이동 중인가?
 └─ YES → SEAD 가능성 높음

주 타겟(하노이 등)으로 직행하지 않고 SAM 인근을 우회/접근?
 └─ YES → SEAD 가능성 확정적

파일럿 보고로 F-105G 또는 ARM 탑재 확인됨?
 └─ YES → SEAD 확정
```

### 폭격기 탐지

아래 패턴 중 하나 이상 해당하면 **폭격기 가능성 있음**으로 처리:

```
목표 방향으로 이동 중인가?
 └─ YES → 폭격기 후보

편대가 선두-후속으로 분리되어 있는가?
 └─ YES → 후속 편대가 폭격기 가능성 높음
           (선두는 escort 가능성)

파일럿 보고로 폭장 확인됨?
 └─ YES → 폭격기 확정
```

**불확실한 경우: 더 위험한 쪽으로 가정.** 폭격기일 수도 있으면 폭격기로 취급한다.

## 무시해도 되는 편대

아래 패턴이면 위협 임무가 아닌 것으로 판단 → MiG 투입 불필요:

```
목표를 향하지 않고 선회 대기 중 → CAP / MiGCAP
후방에서 전진하지 않음 → Jamming / 대기
폭격 완료 후 귀환 방향 이동 중 → 이미 늦음, 무시
```

# 2단계: 위협 우선순위

**고정 우선순위 (점수 무관하게 적용):**

```
1순위: SEAD 확정 또는 가능성 편대
2순위: Strike / Bombing 편대
─────────────────────────────
이하는 MiG 투입 대상 아님:
 - CAP / MiGCAP / Armed Escort → 견제 편대만 배정
 - Jamming / Recon / CSAR → 무시
```

같은 순위 안에서 여러 편대가 있을 경우 아래 점수로 정렬. **최고 점수 = 1차 공격 목표.**

| 조건 | 점수 |
|---|---|
| 폭장 / ARM 확인됨 (파일럿 보고) | +3 |
| SAM 방향으로 이동 중 (SEAD 추정) | +3 |
| 목표 방향으로 이동 중 (미폭격) | +2 |
| 편대 분리 상태 (폭격기와 escort가 떨어짐) | +2 |
| 피해 기체 포함 추정 (저속 / 이탈 행동) | +2 |
| 폭격 완료 후 귀환 중 | -3 (이미 늦음) |
| SAM이 이미 표적 획득 중 | -2 (SAM에 위임) |

---

# 3단계: Escort 평가

폭격기를 공격하기 전, 그 주변 escort 상황을 파악한다.

```
1차 목표 편대 주변에 동반 이동 중인 다른 편대 있는가?
 ├─ NO  → Escort 없음 → 직접 공격 가능 → 5단계로
 └─ YES → Escort 존재 → 아래 판단
```

```
Escort가 MiG보다 폭격기에 더 가까운가?
 ├─ YES → 직접 공격 위험 → 견제 편대 필요
 └─ NO  → MiG가 먼저 접근 가능 → 직접 공격 가능
```

```
가용 MiG 편대 수 ≥ 2인가?
 ├─ YES → 1편대 Escort 견제, 1편대 폭격기 공격
 └─ NO  → 아래 판단

  Escort와 폭격기가 분리되어 있는가?
   ├─ YES → 폭격기 직접 공격
   └─ NO  → 접근 각도 조정 (우회 기동) 또는 대기
```

**견제 편대 운용 원칙**: 견제 편대는 Escort와 실제 교전을 피하고, 기동만으로 Escort의 주의를 분산시킨다.

---

# 4단계: 방공망 확인

```
요격 예정 구역이 SAM 사정권 내인가?
 ├─ YES → SAM과 협조 가능
 │         SAM 발사 중이면 → MiG는 대기 후 회피 기동 중 추가 공격
 │         SAM 미발사 중이면 → MiG 선공격 가능
 └─ NO  → SAM 지원 없음

  AAA 범위 내인가?
   ├─ YES → 제한적 엄호 있음 → 공격 허가
   └─ NO  → 5단계 조건으로 단독 판단
```

---

# 5단계: 출격 결정

아래 조건 점수로 출격 여부를 결정:

| 조건 | 점수 |
|---|---|
| 1차 목표가 폭장 상태로 추정됨 | +2 |
| 후방 또는 측방 접근 가능 | +2 |
| SAM / AAA 엄호 범위 내 | +1 |
| Escort 견제 편대 배정 완료 | +1 |
| 편대 분리 확인됨 | +1 |
| MiG가 현재 미탐지 상태 | +1 |
| Escort와 폭격기가 붙어있고 견제 편대 없음 | -2 |
| SAM이 이미 해당 편대 표적 획득 중 | -2 |
| 방공망 범위 밖 + 후방 접근 불가 | -2 |

```
합계 ≥ 3 → 출격 명령
합계 1~2 → MiG-21 단독 선제 공격만 허가
합계 ≤ 0 → 대기
```

# 6단계: 기종별 전술 지령

## MiG-17F
- 저고도 진입 → 근접 기동전 유도
- SAM/AAA 범위 인접 지역으로 유인 기동 병행 가능
- 교전 후 SAM/AAA 방어망 방향으로 이탈

## MiG-19S
- 중속 기동 + 기관포 공격
- MiG-17과 협조: 17이 교란, 19가 측방 공격
- 단독 운용 시 측방 접근 공격 권장

## MiG-21 (전 계열)
- 고속 단일 공격 → 즉시 이탈 (1 pass only)
- 접근-발사-이탈 전 과정 최단시간에 완료
- 기종 노출 최소화
- 재공격은 연료 마커 기준으로 판단

---

# 7단계: 공격 후 행동

```
공격 완료.

Escort가 MiG를 추적 중인가?
 ├─ YES → 즉시 이탈. SAM/AAA 방향으로 유인 가능
 └─ NO  → 연료 확인

연료 마커 2개 이하인가?
 ├─ YES → 귀환
 └─ NO  → 재공격 검토

재공격:
 1차 목표가 아직 폭격 전인가?
  ├─ YES → 재접근 허가
  └─ NO  → 귀환
```

# 전체 흐름 요약

```
레이더 판독
 → 폭격기 찾기 (이동 방향 + 편대 구조 + 파일럿 보고)
        ↓
우선순위 점수 산정
 → 최고 점수 = 1차 목표
        ↓
Escort 평가
 → 견제 편대 배분 또는 우회 기동
        ↓
방공망 확인
 → SAM/AAA 협조 여부
        ↓
출격 결정 점수
 ≥3: 출격 / 1~2: MiG-21만 / ≤0: 대기
        ↓
기종별 전술 지령
 MiG-17: 기동전 유도
 MiG-19: 측방 협조
 MiG-21: 고속 단일 공격
        ↓
공격 후 이탈 / 귀환
```

# 선택 옵션

## 옵션 A: Fake Scramble (기만 출격)
가용 MiG 편대 2개 이상일 때 사용 가능.
```
1편대 출격 → 교전 없이 Escort 주의 분산
→ 나머지 편대로 폭격기 공격
```

## 옵션 B: B-52 / 대형 폭격 편대 우선
Linebacker 시나리오 전용.
```
B-52 탐지 → 전 편대 B-52 방향 배정
다른 목표 일시 무시
```

# 정보 판단 오류 처리

GCI 판단이 틀릴 수 있다. 별도 판정 없이 상황에 따라 자연스럽게 적용:

- **CAP을 폭격기로 오인**: 파일럿 접촉 보고로 정정 → 다음 이동 페이즈에 목표 재지정
- **폭격기를 CAP으로 오인**: 폭격 완료 후 인식 → 이미 늦음, 귀환 또는 차선 목표
- **더미 편대를 실제로 취급**: 접촉 전까지 실제 편대로 간주
