Keraunos Downfall v.1.0 - 플레이 순서

플레이 순서 = 행동 라운드 (활성 진영 = 최저 주도권, 동점 = 아래쪽 마커)

  1. 날씨 업데이트 [6.1] (날씨 <= 자신의 주도권인 경우)
    1. 날씨를 = 자신의 주도권으로 전진 (자신의 주도권보다 낮은 경우)
    2. 날씨를 2D6만큼 전진 (수정 불가) 하지만 최고 주도권을 초과하지 않음
    3. 날씨 >#91인 경우 → 날씨 이동 완료 후, 새 턴 수행 [12]:

    새로운 날씨 기간에 진입한 경우 (>#01 / >#31 / >#61) → 현재 날씨에 맞춰 날씨 카드를 조정

    1. 턴 마커 전진
    2. 현재 날씨에 맞춰 날씨 카드를 조정
    3. 행동 카드 확인: 이 턴에 들어올 카드가 있으면 카드 더미에 추가 & 섞기
    4. 현재 턴의 증원을 → 대기 영역에 배치
    5. 전체 이벤트 버림 더미 섞기 (최상단 소비 이벤트 포함) → 이벤트 추출 더미 하단에 배치
    6. 공군 조정: 버림 더미의 공군 기호, 현재 & 대기 이벤트에 따라 [10.3] (소련>OKH>OKW>서부)

        7. 소유자(들)는 초과 공군 유닛 제거 (더 이상 사용 불가인 경우)

        소유자(들)는 공군 유닛 추가 [10.3] (새로 제공되는 경우): 포스 풀 → 지도 (사용 가능한 경우)

    7. 모든 명령 배치: 명령 칸 → 추출 컵 (계획 칸 / 행동 트랙 명령은 그대로 유지)
  2. 행동 트랙 채우기 [6.2] (빈 공간이 있고 컵에 명령이 있는 경우): 명령을 아래로 밀기 > 행동 트랙 재충전
  3. 명령 선택 [6.3]:
    일치하는 명령 칸이 비어 있고 계획 중이 아닌 경우 명령 선택 가능 (양면인 경우 = 어느 면이든 선택)
    추축 진영은 추축 이념의 명령 선택 가능 → 자신의 진영 면으로 뒤집기
    양면인 경우 = 대체 면의 일치하는 빈 명령 칸인 경우 뒤집기 가능
    모든 명령: 행동 트랙 → 추출 컵 > 행동 트랙 재충전
    HQ 명령 → 1개의 작전 명령 선택 & 비용 지불
    주도권 명령이 통과된 일치하는 진영 이벤트 공간인 경우: 대기 이벤트 → 주도권으로 뒤집기
    1. 진영의 계획 칸에 있는 명령 선택 (있는 경우) → 일치하는 빈 명령 칸에 배치 (#∆)
    2. 행동 트랙의 명령 선택 → 빈 진영의 계획 칸에 배치 (1∆) / 일치하는 명령 칸 (#∆)
    3. +1∆ / 진영이 플레이할 수 있는 행동 트랙의 "낮은" 명령 (예: 2 추축 징집 = 1/OKW/OKH로 +1∆만)
    4. 행동 트랙에 자신의 이념 명령이 없고 추출 컵에 1+ 명령 → 명령 초기화 [7.12] (3∆)
    5. 행동 트랙에 자신의 이념 명령이 없고 추출 컵에 0 명령 → 자신의 주도권 = 2번째 최저, VI)로 이동
  4. 명령의 주도권 비용 지불 [6.4] (위 선택 / 명령당) → 자신의 주도권 / ∆ 전진
  5. 명령 수행 [6.5] → 명령 참조
  6. 이벤트 해결 [6.6; 10] (대기 이벤트 면에 주도권이 있는 각 진영; 최저에서 최고 주도권 순)
    1. 이벤트 칸의 이벤트를 → 버림 더미 최상단으로 이동 ("소비 이벤트")
    2. 추출 더미 칸의 이벤트를 → 이벤트 칸으로 이동 ("현재 이벤트")
    3. 소유자(들)는 초과 공군 유닛 제거 [10.3] ("소비 이벤트" + "현재 이벤트"에서 더 이상 제공되지 않는 경우)
    4. "현재 이벤트" 카드의 텍스트 해결 → 이벤트 참조
    5. 추출 더미 최상단 카드 공개 (뒤집기) ("대기 이벤트")
    6. EXC: 공개된 카드가 적용 가능한 제거 조건이 있는 경우 → 제거하고 나열된 "대기 이벤트"로 교체
    7. 진영은 "대기 이벤트" 카드당 추가 행동 카드 추출, 카드에 표시된 순서대로
    8. 소유자(들)는 공군 유닛 추가 [10.3] ("소비" + "현재" + "대기 이벤트"에서 새로 제공되는 경우): 포스 풀 → 지도

    추축 공군 기호 공개 → 이벤트를 공개한 플레이어와 일치하는 OKW/OKH 마커 배치 [10.3.1]

    1. 대기 이벤트 → 주도권으로 뒤집기

    첫 번째 공군 유닛 배치는 비엘리트여야 함 [10.3.2]
    육지 영역(서부만) 또는 공급(이벤트 참조) 도시/철도 헥스에 배치 & EZOC인 경우 자신의 유닛이 있어야 함
    자신의 함대가 있는 해역을 통해 추적 가능; 항구-항구; 경합 적 해상 요격(SEA 참조)
